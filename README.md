# LobbySystem
*Minecraft Version: 1.8*  
**LobbySystem** is a simple system used to dynamically add Waypoints, a Game Menu and other convenient Features to let players coordinate through your server.

## Features
- Warps *Create new warps with many different options*
- Compass *Use the compass to teleport to warps*
- DoubleJump *Players can double jump around in the lobby*
- and many more convenient features!

## Commands  

- /system *Shows all commands*
- /system setspawn *Sets the spawnpoint of the lobby*
- /system setwarp <Warp> <Item> <Name> <Slot> *adds a new warp*
- /system infowarp <Warp> *shows information of a specific warp*
- /system removewarp <Warp> *deletes a warp*

## Installation  

1. Download the plugin `LobbySystem.jar` file  
2. Place it in your server's `plugins` folder  
3. Restart or reload your server  
4. Done!
